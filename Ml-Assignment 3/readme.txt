Title: Colon Cancer Nuclei Classification



Number of Instances: 6523

Number of Attributes : 325

 

Dataset description:



This dataset (CNN features) is generated from a Convolutional Neural Network, which is trained on a routine colon cancer nuclei images. 
It has totally 6523 instances, 325 attributes, and 4 classes. The number of observations for each class is not balanced. 
The ground truth (class label) are given in the last column of every row.





Usage: 


Training : 80% (5218)


Testing : 20%  (1305)
